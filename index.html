Principles

Lab values

Every tool should be clear, tactile, useful and alive – with a strong character and a direct creative purpose.

01

Tactile first

Buttons, knobs, sliders and interfaces should feel like real working controls, not paperwork.

02

Fast results

Tools should give a useful result quickly and let the user stay in the creative flow.

03

Character wins

Noise, imperfection, movement and attitude are part of the product, not a defect.

04

Useful experiments

Every experiment should become practical, useful or emotionally memorable.
